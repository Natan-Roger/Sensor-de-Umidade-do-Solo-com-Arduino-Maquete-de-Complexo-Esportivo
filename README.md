# Sensor-de-Umidade-do-Solo-com-Arduino-Maquete-de-Complexo-Esportivo
Sistema de sensor de umidade do solo com Arduino, prototipado sobre uma maquete de complexo esportivo (pista de atletismo e quadras) construída do zero com corte a laser profissional. Projeto do técnico em IoT (Senac) voltado ao monitoramento de áreas verdes e futura automação de irrigação.
Projeto desenvolvido durante o curso técnico em Internet das Coisas (IoT) no Senac, com o objetivo de aplicar, na prática, os conceitos de sensoriamento e automação em um cenário real: o monitoramento de umidade do solo em áreas verdes de um complexo esportivo.
Para representar o cenário de aplicação, foi construída do zero uma maquete física de um complexo esportivo (pista de atletismo, quadras poliesportivas e área verde), utilizando corte a laser profissional para as peças estruturais. O sistema eletrônico foi montado sobre a maquete com Arduino e sensores de umidade do solo, simulando como a solução funcionaria em um ambiente real de gestão de áreas verdes.
⚙️ Como funciona
	•	O sensor de umidade do solo é inserido na terra e envia o sinal analógico para o Arduino.
	•	O Arduino lê o valor e o converte em um índice de umidade.
	•	Com base nesse índice, o sistema identifica se o solo está seco ou úmido, servindo como base para uma futura automação de irrigação.
	•	Toda a lógica foi prototipada em protoboard, com o circuito montado e testado sobre a própria maquete.
